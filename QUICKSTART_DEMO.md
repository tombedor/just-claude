# Just-Claude Quickstart Demo

This is a complete walkthrough showing just-claude in action.

```bash

mkdir my-awesome-project
cd my-awesome-project

# Create a justfile with some common tasks

    echo "Setting up dependencies..."

# Initialize npm

```

## Step 2: Install just-claude

```bash
npm install just-claude
```

**What happens:**
- Creates `.claude/hooks/detect-justfile.sh`
- Configures `.claude/settings.json` with SessionStart hook
- Backs up existing settings if present

## Step 7: Use with Claude Code

Now start Claude Code in your project:

```bash
claude
```

### Example Interactions

**You:** "Can you run the tests?"

**Claude:** *Sees the `just-test` skill and runs:*
```bash
just test
```

**You:** "Deploy this to staging"

**Claude:** *Sees `just-deploy` requires an environment parameter:*
```bash
just deploy staging
```

**You:** "Start the dev server on port 8080"

**Claude:** *Uses the optional port parameter:*
```bash
just dev 8080
```

**You:** "Format the code"

**Claude:** *Runs the format recipe:*
```bash
just fmt
```

## Advanced: With Modules

Create a justfile with modules:

```bash
# Main justfile
cat > justfile << 'EOF'
# Build main application
build:
    echo "Building main app..."

mod backend
mod frontend
EOF

# Backend module
mkdir -p backend
cat > backend/justfile << 'EOF'
# Deploy backend services
deploy:
    echo "Deploying backend..."

# Run backend tests
test:
    echo "Testing backend..."
EOF

# Frontend module
mkdir -p frontend
cat > frontend/justfile << 'EOF'
# Build frontend assets
build:
    echo "Building frontend..."

# Deploy frontend
deploy:
    echo "Deploying frontend..."
EOF

# Regenerate skills
npx just-claude regenerate
```

**Skills created:**
```
- just-build              (main)
- just-backend/deploy     (module)
- just-backend/test       (module)
- just-frontend/build     (module)
- just-frontend/deploy    (module)
```

Claude can now invoke:
- `just build` (main)
- `just backend::deploy` (module)
- `just frontend::build` (module)

## Cleanup

To uninstall:

```bash
npm uninstall just-claude
```

This removes:
- Hook script (`.claude/hooks/detect-justfile.sh`)
- Generated skills (`.claude/skills/just-*`)
- Hook configuration from settings.json
- Restores backup if no other hooks exist
